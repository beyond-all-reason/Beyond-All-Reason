Skybox Library for Terraform Brush
===================================

This folder is scanned by the Terraform Brush environment panel to populate
the skybox selector.  Place cubemap DDS files here and they will appear
automatically in the UI.

To get skyboxes:
  → Discord #mapping channel pins (Beyond All Reason server)

Place .dds cubemap files in this folder:
  luaui/RmlWidgets/gui_terraform_brush/skyboxes/

Optional: add a preview image with the same base name + " - Preview.jpg"
for thumbnails in the UI (e.g. "MySkybox.dds" + "MySkybox - Preview.jpg").
